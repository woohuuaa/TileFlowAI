# TileFlowAI Roadmap

## Phase 0

- Confirm design rules with questionnaire responses
- Collect representative DWG samples
- Lock the first supported AutoCAD version

## Phase 1

- Implement geometry primitives
- Support rectangular room prototype
- Build candidate generation for `0°` and `90°`

## Phase 2

- Add cut-tile scoring and tie-break rules
- Add conservative order estimation
- Add report-ready summaries

## Phase 3

- Connect the engine to AutoCAD commands
- Add preview entities and preview-layer cleanup
- Support final layout draw-back into a dedicated layer

## Phase 4

- Add deterministic CAD object detection rules
- Add user confirmation flow for exclusions
- Add agent-assisted classification for ambiguous objects

## Phase 5

- Improve performance on irregular spaces
- Add more layout modes
- Consider leftover reuse optimization
