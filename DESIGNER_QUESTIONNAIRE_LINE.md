# TileFlowAI 設計師需求確認問卷（LINE 精簡版）

可直接貼給設計師：

```text
嗨，我們在整理 TileFlowAI 的需求，想先確認幾個 CAD 與排磚規則：

1. 圖面單位是什麼？mm / cm / m
2. 鋪貼區域通常有封閉 polyline 嗎？如果沒有，房間邊界一般怎麼判斷？
3. 牆、門、柱、設備、櫃體有固定的 layer 或 block 名稱嗎？如果有請提供常用名稱。
4. 哪些固定物件要從鋪貼面積中扣掉？哪些只作為排版參考、不扣面積？
5. 門片、門框、門檻會不會影響排磚？是要扣面積，還是只當起鋪參考？
6. 第一版是否只需要支援長方形磁磚，並只比較直貼 / 橫貼？
7. 縫寬要不要算進去？常用縫寬是多少？
8. 你們判斷最佳方案最在意什麼？碎磚最少、最小碎片不能太小、廢料最少、還是視覺對稱？
9. 下單結果要看哪些數字？片數、箱數、備料率？
10. 能不能提供 2-3 份代表性的 DWG 圖，最好包含一份規則清楚、一份比較複雜的案例？
```
